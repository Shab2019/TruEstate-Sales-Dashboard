# TruEstate Retail Backend

This is the backend for the Retail Sales Management System developed for the TruEstate SDE Internship Assignment.

## Features

- Loads the provided retail sales CSV dataset at startup.
- Exposes `/api/sales` endpoint supporting:
  - Search
  - Filters (multi-select + range)
  - Sorting
  - Pagination
- `/api/sales/filters` returns all available filter options.
